# Contributing

We welcome contributions!

## Rules

- All changes must go through a Pull Request
- PRs must be approved before merging
- By contributing, you agree to the CLA

## Guidelines

- Keep changes focused and small
- Follow project coding standards
- Add tests when possible

## Process

1. Fork the repo
2. Create a branch
3. Open a PR
4. Sign the CLA
5. Wait for review

## Governance

This project is open source and maintained by Alessio Leodori.

We welcome contributions, but the core team defines the roadmap and has final say on all changes.
